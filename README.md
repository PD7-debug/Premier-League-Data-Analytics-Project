# ⚽ Premier League Data Analysis (2024–25) ⚽

## Project Overview

This project performs a comprehensive data analysis of the **2024–25 English Premier League season** using Python and data science techniques.

The goal is to explore **team performance, player contributions, tactical trends, and league dynamics** using multiple datasets.

---

# Key Analysis Areas

### Match Analysis

* Goal distribution across matches
* Home vs away goal patterns
* Match outcome probabilities

### Team Shooting Analysis

* Shot volume and accuracy
* Conversion rate
* Attacking efficiency

### Player Performance

* Goals per 90
* Assists per 90
* Player dependency analysis

### Goalkeeper Performance

* Save percentage
* Clean sheets
* Defensive impact

### Playing Time Analysis

* Squad rotation
* Substitution intensity
* Minutes concentration

### Discipline & Aggression

* Fouls committed
* Yellow/red cards
* Aggression index

### Opponent Pressure

* Aggression faced by teams
* Defensive workload

### Nationality Representation

* Player nationality distribution
* Global recruitment trends

### Home vs Away Performance

* Home advantage
* Goal scoring patterns

### League Table Analysis

* Points distribution
* xG vs actual goals
* Defensive efficiency

---

# 🛠 Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* VS Code

---

# 📈 Key Insights

* Certain teams show strong **home advantage**, earning significantly more points at home.
* **Aggressive defensive teams** record higher tackle and interception counts.
* Some teams **overperform their expected goals (xG)**, indicating efficient finishing.
* The Premier League demonstrates **high international diversity**, with players from many nationalities contributing significant minutes.
